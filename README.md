# SMRPO projekt

Orodje za podporo razvoju programske opreme po metodi Scrum
 
# GET STARTED

FRONTEND

$ cd ScrumTracker
$ npm install
$ npm install -g @quasar/cli
$ quasar dev

-> web aplikacija se odpre v brskalniku

BACKEND

$ cd backend
$ pip install requirements.txt
$ FLASK_APP=app.py
$ flask run
